Montréal
========
:revnumber: 0.0.1-draft
:toc: preamble

A RISC-V RV32E processor designed for TinyTapeout.

Github repository:
https://github.com/ubc-asic/superscalar-summer/tree/master +
Development branch:
https://github.com/ubc-asic/superscalar-summer/tree/feature/core
